## 2. 云平台客户端技术方案

### 2.1 客户端工厂设计

采用 **工厂模式 + 缓存机制**，按 `clusterId` 缓存客户端实例，避免重复创建连接：

```java
@Component
public class CloudClientFactory {

    private final ConcurrentHashMap<String, OSClient.OSClientV3> openStackClients = new ConcurrentHashMap<>();
    private final ConcurrentHashMap<String, CephMgrClient> cephMgrClients = new ConcurrentHashMap<>();
    private final ConcurrentHashMap<String, S3Client> cephRgwClients = new ConcurrentHashMap<>();
    private final ConcurrentHashMap<String, KubernetesClient> k8sClients = new ConcurrentHashMap<>();

    @Autowired
    private ClusterConfigMapper clusterConfigMapper;
    
    @Autowired
    private EncryptUtil encryptUtil;

    /**
     * 获取 OpenStack 客户端
     */
    public OSClient.OSClientV3 getOpenStackClient(String clusterId) {
        return openStackClients.computeIfAbsent(clusterId, id -> {
            ClusterConfig config = getAndDecryptConfig(id);
            return createOpenStackClient(config);
        });
    }

    /**
     * 获取 Ceph MGR 客户端
     */
    public CephMgrClient getCephMgrClient(String clusterId) {
        return cephMgrClients.computeIfAbsent(clusterId, id -> {
            ClusterConfig config = getAndDecryptConfig(id);
            return createCephMgrClient(config);
        });
    }

    /**
     * 获取 Ceph RGW S3 客户端
     */
    public S3Client getCephRgwClient(String clusterId) {
        return cephRgwClients.computeIfAbsent(clusterId, id -> {
            ClusterConfig config = getAndDecryptConfig(id);
            return createCephRgwClient(config);
        });
    }

    /**
     * 获取 Kubernetes 客户端
     */
    public KubernetesClient getKubernetesClient(String clusterId) {
        return k8sClients.computeIfAbsent(clusterId, id -> {
            ClusterConfig config = getAndDecryptConfig(id);
            return createKubernetesClient(config);
        });
    }

    /**
     * 清除指定集群的客户端缓存
     */
    public void invalidateClient(String clusterId) {
        OSClient.OSClientV3 osClient = openStackClients.remove(clusterId);
        if (osClient != null) {
            // openstack4j 客户端无需显式关闭
        }
        
        CephMgrClient cephMgrClient = cephMgrClients.remove(clusterId);
        if (cephMgrClient != null) {
            cephMgrClient.close();
        }
        
        S3Client s3Client = cephRgwClients.remove(clusterId);
        if (s3Client != null) {
            s3Client.close();
        }
        
        KubernetesClient k8sClient = k8sClients.remove(clusterId);
        if (k8sClient != null) {
            k8sClient.close();
        }
    }

    /**
     * 获取并解密集群配置
     */
    private ClusterConfig getAndDecryptConfig(String clusterId) {
        ClusterConfig config = clusterConfigMapper.selectById(clusterId);
        if (config == null) {
            throw new CloudException("CLOUD_0001", "集群配置不存在");
        }
        // 解密敏感字段
        String decryptedJson = encryptUtil.decrypt(config.getConfigJson());
        config.setConfigJson(decryptedJson);
        return config;
    }

    // 创建各类客户端的具体实现...
}
```

```java
private S3Client createCephRgwClient(ClusterConfig config) {
    try {
        CephConfig cephConfig = JSON.parseObject(config.getConfigJson(), CephConfig.class);
        
        AwsBasicCredentials credentials = AwsBasicCredentials.create(
            cephConfig.getAccessKey(), 
            cephConfig.getSecretKey()
        );

        return S3Client.builder()
            .endpointOverride(URI.create(cephConfig.getRgwEndpoint()))
            .credentialsProvider(StaticCredentialsProvider.create(credentials))
            .region(Region.of("us-east-1"))  // RGW 不校验 region，但 SDK 要求
            .serviceConfiguration(S3Configuration.builder()
                .pathStyleAccessEnabled(true)  // RGW 使用 path-style
                .build())
            .build();
    } catch (Exception e) {
        throw new CloudException("CEPH_0004", "RGW 客户端创建失败: " + e.getMessage());
    }
}
```

```java
private KubernetesClient createKubernetesClient(ClusterConfig config) {
    try {
        K8sConfig k8sConfig = JSON.parseObject(config.getConfigJson(), K8sConfig.class);
        
        Config fabric8Config;
        
        if ("IN_CLUSTER".equals(k8sConfig.getConnectionType())) {
            // 使用集群内配置
            fabric8Config = Config.autoConfigure(null);
        } else {
            // 使用 kubeconfig
            String kubeconfigContent = new String(
                Base64.getDecoder().decode(k8sConfig.getKubeconfigContent()), 
                StandardCharsets.UTF_8
            );
            fabric8Config = Config.fromKubeconfig(kubeconfigContent);
        }
        
        return new KubernetesClientBuilder()
            .withConfig(fabric8Config)
            .build();
    } catch (Exception e) {
        throw new CloudException("K8S_0005", "Kubernetes 客户端创建失败: " + e.getMessage());
    }
}
```

## 3. 敏感信息加密方案

### 3.1 AES-256-GCM 加密实现

```java
@Component
public class EncryptUtil {

    @Value("${opencloud.encrypt.key}")
    private String encryptKey;

    private static final String ALGORITHM = "AES/GCM/NoPadding";
    private static final int GCM_IV_LENGTH = 12;
    private static final int GCM_TAG_LENGTH = 128;

    /**
     * 加密
     */
    public String encrypt(String plaintext) {
        try {
            byte[] iv = new byte[GCM_IV_LENGTH];
            SecureRandom random = new SecureRandom();
            random.nextBytes(iv);

            Cipher cipher = Cipher.getInstance(ALGORITHM);
            GCMParameterSpec parameterSpec = new GCMParameterSpec(GCM_TAG_LENGTH, iv);
            cipher.init(Cipher.ENCRYPT_MODE, getSecretKey(), parameterSpec);

            byte[] encrypted = cipher.doFinal(plaintext.getBytes(StandardCharsets.UTF_8));
            
            // IV + ciphertext
            ByteBuffer byteBuffer = ByteBuffer.allocate(iv.length + encrypted.length);
            byteBuffer.put(iv);
            byteBuffer.put(encrypted);
            
            return Base64.getEncoder().encodeToString(byteBuffer.array());
        } catch (Exception e) {
            throw new RuntimeException("加密失败", e);
        }
    }

    /**
     * 解密
     */
    public String decrypt(String ciphertext) {
        try {
            byte[] decoded = Base64.getDecoder().decode(ciphertext);
            
            ByteBuffer byteBuffer = ByteBuffer.wrap(decoded);
            byte[] iv = new byte[GCM_IV_LENGTH];
            byteBuffer.get(iv);
            byte[] encrypted = new byte[byteBuffer.remaining()];
            byteBuffer.get(encrypted);

            Cipher cipher = Cipher.getInstance(ALGORITHM);
            GCMParameterSpec parameterSpec = new GCMParameterSpec(GCM_TAG_LENGTH, iv);
            cipher.init(Cipher.DECRYPT_MODE, getSecretKey(), parameterSpec);

            byte[] decrypted = cipher.doFinal(encrypted);
            return new String(decrypted, StandardCharsets.UTF_8);
        } catch (Exception e) {
            throw new RuntimeException("解密失败", e);
        }
    }

    private SecretKey getSecretKey() {
        byte[] keyBytes = encryptKey.getBytes(StandardCharsets.UTF_8);
        // 确保密钥长度为 32 字节（256 位）
        byte[] keyBytes32 = Arrays.copyOf(keyBytes, 32);
        return new SecretKeySpec(keyBytes32, "AES");
    }
}
```

## 4. 配额检查技术实现

### 4.1 配额检查服务

```java
@Service
public class QuotaCheckService {

    @Autowired
    private CloudQuotaMapper quotaMapper;
    
    @Autowired
    private CloudClientFactory clientFactory;
    
    @Autowired
    private StringRedisTemplate redisTemplate;

    private static final String QUOTA_CACHE_KEY = "cloud:quota:%s";
    private static final String USAGE_CACHE_KEY = "cloud:usage:%s:%s";
    private static final long CACHE_TTL_MINUTES = 5;

    /**
     * 检查 OpenStack 配额
     */
    public void checkOpenStackQuota(String tenantId, String clusterId, 
                                     QuotaRequest request) {
        CloudQuota quota = getQuota(tenantId);
        OpenStackUsage usage = getOpenStackUsage(tenantId, clusterId);

        // 检查 vCPU
        if (quota.getOpenstackVcpuLimit() >= 0) {
            int totalVcpu = usage.getVcpuUsed() + request.getVcpus();
            if (totalVcpu > quota.getOpenstackVcpuLimit()) {
                throw new CloudException("CLOUD_0104", 
                    String.format("vCPU 配额不足: 已使用 %d, 申请 %d, 限制 %d",
                        usage.getVcpuUsed(), request.getVcpus(), quota.getOpenstackVcpuLimit()));
            }
        }

        // 检查内存
        if (quota.getOpenstackMemoryLimit() >= 0) {
            long totalMemory = usage.getMemoryUsed() + request.getRam();
            if (totalMemory > quota.getOpenstackMemoryLimit()) {
                throw new CloudException("CLOUD_0104", "内存配额不足");
            }
        }

        // 检查实例数
        if (quota.getOpenstackInstanceLimit() >= 0) {
            int totalInstances = usage.getInstanceUsed() + request.getCount();
            if (totalInstances > quota.getOpenstackInstanceLimit()) {
                throw new CloudException("CLOUD_0104", "实例数配额不足");
            }
        }
    }

    /**
     * 获取配额配置（带缓存）
     */
    private CloudQuota getQuota(String tenantId) {
        String cacheKey = String.format(QUOTA_CACHE_KEY, tenantId);
        String cached = redisTemplate.opsForValue().get(cacheKey);
        
        if (cached != null) {
            return JSON.parseObject(cached, CloudQuota.class);
        }

        CloudQuota quota = quotaMapper.selectByTenantId(tenantId);
        if (quota == null) {
            // 默认配额为 0（拒绝创建）
            quota = new CloudQuota();
            quota.setOpenstackVcpuLimit(0);
            quota.setOpenstackMemoryLimit(0L);
            quota.setOpenstackInstanceLimit(0);
        }

        redisTemplate.opsForValue().set(cacheKey, JSON.toJSONString(quota), 
            CACHE_TTL_MINUTES, TimeUnit.MINUTES);
        return quota;
    }

    /**
     * 获取 OpenStack 资源使用量（带缓存）
     */
    private OpenStackUsage getOpenStackUsage(String tenantId, String clusterId) {
        String cacheKey = String.format(USAGE_CACHE_KEY, tenantId, clusterId);
        String cached = redisTemplate.opsForValue().get(cacheKey);
        
        if (cached != null) {
            return JSON.parseObject(cached, OpenStackUsage.class);
        }

        // 查询云平台统计使用量
        OSClient.OSClientV3 client = clientFactory.getOpenStackClient(clusterId);
        
        List<? extends Server> servers = client.compute().servers().list();
        int vcpuUsed = 0;
        long memoryUsed = 0;
        int instanceUsed = servers.size();

        for (Server server : servers) {
            Flavor flavor = client.compute().flavors().get(server.getFlavorId());
            if (flavor != null) {
                vcpuUsed += flavor.getVcpus();
                memoryUsed += flavor.getRam();
            }
        }

        OpenStackUsage usage = new OpenStackUsage();
        usage.setVcpuUsed(vcpuUsed);
        usage.setMemoryUsed(memoryUsed);
        usage.setInstanceUsed(instanceUsed);

        redisTemplate.opsForValue().set(cacheKey, JSON.toJSONString(usage), 
            CACHE_TTL_MINUTES, TimeUnit.MINUTES);
        return usage;
    }

    /**
     * 清除使用量缓存（资源变更后调用）
     */
    public void evictUsageCache(String tenantId, String clusterId) {
        String cacheKey = String.format(USAGE_CACHE_KEY, tenantId, clusterId);
        redisTemplate.delete(cacheKey);
    }
}
```

## 5. 异步任务技术实现

### 5.1 任务服务

```java
@Service
public class TaskService {

    @Autowired
    private CloudTaskMapper taskMapper;
    
    @Autowired
    private StringRedisTemplate redisTemplate;

    private static final String TASK_CACHE_KEY = "cloud:task:%s";
    private static final long TASK_CACHE_TTL_DAYS = 7;

    /**
     * 创建任务
     */
    @Transactional
    public CloudTask createTask(String tenantId, TaskType type, 
                                 ResourceType resourceType, String clusterId,
                                 String createdBy) {
        CloudTask task = new CloudTask();
        task.setTaskId(UUID.randomUUID().toString());
        task.setTenantId(tenantId);
        task.setType(type.name());
        task.setStatus(TaskStatus.PENDING.name());
        task.setResourceType(resourceType.name());
        task.setClusterId(clusterId);
        task.setProgress(0);
        task.setCreatedBy(createdBy);
        task.setCreatedAt(LocalDateTime.now());

        taskMapper.insert(task);
        cacheTask(task);

        return task;
    }

    /**
     * 开始执行任务
     */
    @Transactional
    public void startTask(String taskId) {
        LambdaUpdateWrapper<CloudTask> update = new LambdaUpdateWrapper<>();
        update.eq(CloudTask::getTaskId, taskId)
              .set(CloudTask::getStatus, TaskStatus.RUNNING.name())
              .set(CloudTask::getStartedAt, LocalDateTime.now());
        taskMapper.update(null, update);
        
        evictTaskCache(taskId);
    }

    /**
     * 更新任务进度
     */
    @Transactional
    public void updateProgress(String taskId, int progress) {
        LambdaUpdateWrapper<CloudTask> update = new LambdaUpdateWrapper<>();
        update.eq(CloudTask::getTaskId, taskId)
              .set(CloudTask::getProgress, Math.min(100, Math.max(0, progress)));
        taskMapper.update(null, update);
    }

    /**
     * 完成任务
     */
    @Transactional
    public void completeTask(String taskId, Object result) {
        LambdaUpdateWrapper<CloudTask> update = new LambdaUpdateWrapper<>();
        update.eq(CloudTask::getTaskId, taskId)
              .set(CloudTask::getStatus, TaskStatus.SUCCESS.name())
              .set(CloudTask::getProgress, 100)
              .set(CloudTask::getResultJson, JSON.toJSONString(result))
              .set(CloudTask::getCompletedAt, LocalDateTime.now());
        taskMapper.update(null, update);
        
        evictTaskCache(taskId);
    }

    /**
     * 任务失败
     */
    @Transactional
    public void failTask(String taskId, String errorMsg) {
        LambdaUpdateWrapper<CloudTask> update = new LambdaUpdateWrapper<>();
        update.eq(CloudTask::getTaskId, taskId)
              .set(CloudTask::getStatus, TaskStatus.FAILED.name())
              .set(CloudTask::getErrorMsg, errorMsg)
              .set(CloudTask::getCompletedAt, LocalDateTime.now());
        taskMapper.update(null, update);
        
        evictTaskCache(taskId);
    }

    /**
     * 获取任务（带缓存）
     */
    public CloudTask getTask(String taskId) {
        String cacheKey = String.format(TASK_CACHE_KEY, taskId);
        String cached = redisTemplate.opsForValue().get(cacheKey);
        
        if (cached != null) {
            return JSON.parseObject(cached, CloudTask.class);
        }

        CloudTask task = taskMapper.selectByTaskId(taskId);
        if (task != null) {
            cacheTask(task);
        }
        return task;
    }

    private void cacheTask(CloudTask task) {
        String cacheKey = String.format(TASK_CACHE_KEY, task.getTaskId());
        redisTemplate.opsForValue().set(cacheKey, JSON.toJSONString(task),
            TASK_CACHE_TTL_DAYS, TimeUnit.DAYS);
    }

    private void evictTaskCache(String taskId) {
        String cacheKey = String.format(TASK_CACHE_KEY, taskId);
        redisTemplate.delete(cacheKey);
    }
}
```

### 5.2 异步执行器

```java
@Service
public class AsyncTaskExecutor {

    @Autowired
    private TaskService taskService;
    
    @Autowired
    private CloudClientFactory clientFactory;
    
    @Autowired
    private QuotaCheckService quotaCheckService;

    @Async("taskExecutor")
    public void executeInstanceCreate(String taskId, InstanceCreateRequest request) {
        try {
            taskService.startTask(taskId);
            
            OSClient.OSClientV3 client = clientFactory.getOpenStackClient(request.getClusterId());
            
            // 构建创建参数
            ServerCreateBuilder builder = Builders.server()
                .name(request.getName())
                .flavor(request.getFlavorId())
                .image(request.getImageId())
                .networks(request.getNetworkIds().stream()
                    .map(id -> UUID.fromString(id))
                    .collect(Collectors.toList()));
            
            if (request.getKeyPairName() != null) {
                builder.keypairName(request.getKeyPairName());
            }

            // 创建云主机
            Server server = client.compute().servers().boot(builder.build());
            String instanceId = server.getId();
            
            // 轮询状态
            int maxAttempts = 60;  // 最多等待 10 分钟
            for (int i = 0; i < maxAttempts; i++) {
                Thread.sleep(10000);  // 10 秒轮询一次
                
                Server current = client.compute().servers().get(instanceId);
                String status = current.getStatus().name();
                
                taskService.updateProgress(taskId, Math.min(90, (i + 1) * 100 / maxAttempts));
                
                if ("ACTIVE".equals(status)) {
                    taskService.completeTask(taskId, 
                        Map.of("instanceId", instanceId, "name", current.getName()));
                    quotaCheckService.evictUsageCache(request.getTenantId(), request.getClusterId());
                    return;
                } else if ("ERROR".equals(status)) {
                    taskService.failTask(taskId, "云主机创建失败，状态为 ERROR");
                    return;
                }
            }
            
            taskService.failTask(taskId, "云主机创建超时");
            
        } catch (Exception e) {
            log.error("创建云主机任务执行失败", e);
            taskService.failTask(taskId, e.getMessage());
        }
    }
}
```

## 6. 定时资源同步实现

```java
@Component
public class ResourceSyncJob {

    @Autowired
    private ClusterConfigMapper clusterConfigMapper;
    
    @Autowired
    private CloudSyncLogMapper syncLogMapper;
    
    @Autowired
    private CloudClientFactory clientFactory;
    
    @Autowired
    private StringRedisTemplate redisTemplate;

    private static final String SYNC_LOCK_KEY = "cloud:sync:%s";

    /**
     * 定时同步所有集群（每 5 分钟）
     */
    @Scheduled(fixedRate = 5 * 60 * 1000)
    public void syncAllClusters() {
        List<ClusterConfig> clusters = clusterConfigMapper.selectList(
            new LambdaQueryWrapper<ClusterConfig>()
                .eq(ClusterConfig::getStatus, "ACTIVE")
                .eq(ClusterConfig::getDeleted, 0)
        );

        for (ClusterConfig cluster : clusters) {
            try {
                syncCluster(cluster, "AUTO");
            } catch (Exception e) {
                log.error("同步集群 {} 失败", cluster.getId(), e);
            }
        }
    }

    /**
     * 同步指定集群
     */
    public void syncCluster(ClusterConfig cluster, String syncType) {
        String lockKey = String.format(SYNC_LOCK_KEY, cluster.getId());
        
        // 尝试获取锁（防止并发同步）
        Boolean locked = redisTemplate.opsForValue()
            .setIfAbsent(lockKey, "1", 10, TimeUnit.MINUTES);
        
        if (!Boolean.TRUE.equals(locked)) {
            log.warn("集群 {} 正在同步中，跳过本次同步", cluster.getId());
            return;
        }

        CloudSyncLog syncLog = new CloudSyncLog();
        syncLog.setClusterId(cluster.getId());
        syncLog.setTenantId(cluster.getTenantId());
        syncLog.setSyncType(syncType);
        syncLog.setStartedAt(LocalDateTime.now());

        try {
            Map<String, Integer> counts = new HashMap<>();

            switch (cluster.getType()) {
                case "OPENSTACK":
                    counts = syncOpenStack(cluster);
                    break;
                case "CEPH":
                    counts = syncCeph(cluster);
                    break;
                case "KUBERNETES":
                    counts = syncKubernetes(cluster);
                    break;
            }

            syncLog.setStatus("SUCCESS");
            syncLog.setResourceCounts(JSON.parseObject(JSON.toJSONString(counts)));
            
            // 清除使用量缓存
            redisTemplate.delete(String.format("cloud:usage:%s:%s", 
                cluster.getTenantId(), cluster.getId()));

        } catch (Exception e) {
            syncLog.setStatus("FAILED");
            syncLog.setErrorMsg(e.getMessage());
            log.error("同步集群 {} 失败", cluster.getId(), e);
        } finally {
            syncLog.setCompletedAt(LocalDateTime.now());
            syncLogMapper.insert(syncLog);
            redisTemplate.delete(lockKey);
        }
    }

    private Map<String, Integer> syncOpenStack(ClusterConfig cluster) {
        OSClient.OSClientV3 client = clientFactory.getOpenStackClient(cluster.getId());
        
        Map<String, Integer> counts = new HashMap<>();
        counts.put("instances", client.compute().servers().list().size());
        counts.put("volumes", client.blockStorage().volumes().list().size());
        counts.put("networks", client.networking().network().list().size());
        counts.put("images", client.imagesV2().list().size());
        
        return counts;
    }

    private Map<String, Integer> syncCeph(ClusterConfig cluster) {
        CephMgrClient client = clientFactory.getCephMgrClient(cluster.getId());
        
        Map<String, Integer> counts = new HashMap<>();
        counts.put("pools", client.getPools().block().size());
        // 其他统计...
        
        return counts;
    }

    private Map<String, Integer> syncKubernetes(ClusterConfig cluster) {
        KubernetesClient client = clientFactory.getKubernetesClient(cluster.getId());
        
        Map<String, Integer> counts = new HashMap<>();
        counts.put("namespaces", client.namespaces().list().getItems().size());
        counts.put("deployments", client.apps().deployments().inAnyNamespace().list().getItems().size());
        counts.put("pods", client.pods().inAnyNamespace().list().getItems().size());
        
        return counts;
    }
}
```

## 10. 安全加固 Checklist

- [ ] 加密密钥长度 32 字节（256 位），通过配置中心注入，不硬编码
- [ ] 敏感配置（密码、密钥）AES-256-GCM 加密存储

- [ ] 云平台 API 调用超时控制，防止长时间阻塞
- [ ] 客户端缓存定期清理，避免内存泄漏
- [ ] 任务执行超时自动标记失败，避免僵尸任务
- [ ] 同步操作加分布式锁，防止并发冲突
- [ ] 配额检查在创建资源前执行，防止超用
- [ ] 生产环境 HTTPS 强制（由负载均衡处理）

## 11. 异常处理规范

```java
// CloudException 统一结构
@Getter
public class CloudException extends RuntimeException {
    private final String errorCode;
    private final int httpStatus;

    public CloudException(String errorCode, String message) {
        this(errorCode, message, HttpStatus.BAD_REQUEST);
    }

    public CloudException(String errorCode, String message, HttpStatus status) {
        super(message);
        this.errorCode = errorCode;
        this.httpStatus = status.value();
    }
}

// GlobalExceptionHandler 统一返回格式
@ExceptionHandler(CloudException.class)
public ResponseEntity<Result<Void>> handleCloudException(CloudException e) {
    return ResponseEntity
        .status(e.getHttpStatus())
        .body(Result.fail(e.getErrorCode(), e.getMessage()));
}
```

| 系列 | HTTP 状态 | 说明 |
|------|-----------|------|
| CLOUD_0xxx | 400 / 404 / 503 | 集群配置相关 |
| CLOUD_010x | 400 / 403 / 404 | 资源操作相关 |
| OS_xxxx | 400 / 409 / 502 | OpenStack 特定异常 |
| CEPH_xxxx | 404 / 409 / 503 | Ceph 特定异常 |
| K8S_xxxx | 400 / 404 / 503 | Kubernetes 特定异常 |
| TASK_xxxx | 400 / 404 / 500 | 任务相关异常 |
