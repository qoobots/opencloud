## 注意事项

- 各平台客户端实例应**按 cluster_config 缓存**，避免每次请求重新建立连接（使用 `ConcurrentHashMap` + 配置变更时失效）
- OpenStack Token 有效期约 1 小时，需要在 openstack4j 层处理自动刷新
- K8s kubeconfig 文件路径或 in-cluster 模式需在 ClusterConfig 中区分
- Ceph MGR REST API 默认端口 8080，需确认防火墙放行
- 敏感配置（密码、密钥）使用 AES-256-GCM 加密存储
