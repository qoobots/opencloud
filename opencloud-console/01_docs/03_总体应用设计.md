**HTTP 状态码与业务码映射：**

| HTTP 状态码 | 业务码 | 含义 |
|------------|-------|------|
| 200 | 200 | 成功 |
| 400 | 400 | 请求参数错误 |
| 401 | 401 | 未认证/Token 过期 |
| 403 | 403 | 无权限 |
| 404 | 404 | 资源不存在 |
| 423 | 423 | 账号锁定 |
| 429 | 429 | 请求过于频繁 |

### 2.3 JWT 认证过滤器

```java
// JWT 认证过滤器（opencloud-auth）
@Component
@RequiredArgsConstructor
public class JwtAuthenticationFilter extends OncePerRequestFilter {
    
    private final JwtUtil jwtUtil;
    private final RedisTemplate<String, String> redisTemplate;
    
    @Override
    protected void doFilterInternal(HttpServletRequest request, 
                                    HttpServletResponse response, 
                                    FilterChain chain) {
        // 1. 从 Header 提取 Token
        String token = extractToken(request);
        
        // 2. 校验 Token 有效性
        if (token != null && jwtUtil.validateToken(token)) {
            // 3. 检查黑名单
            if (isTokenBlacklisted(token)) {
                throw new BusinessException("Token 已失效");
            }
            
            // 4. 加载用户权限
            Authentication authentication = buildAuthentication(token);
            SecurityContextHolder.getContext().setAuthentication(authentication);
        }
        
        chain.doFilter(request, response);
    }
}
```

## 五、外部服务对接设计

### 5.1 OpenStack 对接

```java
// OpenStack 客户端（opencloud-cloud）
@Component
@RequiredArgsConstructor
public class OpenStackClient {
    
    private final WebClient webClient;
    private final ClusterConfigService configService;
    
    // 获取 Token
    public Mono<String> authenticate(Long clusterId) {
        ClusterConfig config = configService.getById(clusterId);
        return webClient.post()
            .uri(config.getAuthUrl() + "/v3/auth/tokens")
            .bodyValue(buildAuthRequest(config))
            .retrieve()
            .toBodilessEntity()
            .map(response -> response.getHeaders().getFirst("X-Subject-Token"));
    }
    
    // 调用 Nova API
    public Mono<JsonNode> listInstances(Long clusterId) {
        return getToken(clusterId)
            .flatMap(token -> webClient.get()
                .uri(getNovaUrl(clusterId) + "/servers/detail")
                .header("X-Auth-Token", token)
                .retrieve()
                .bodyToMono(JsonNode.class));
    }
}
```

### 5.2 Prometheus 对接

```java
// Prometheus 客户端（opencloud-monitor）
@Component
@RequiredArgsConstructor
public class PrometheusClient {
    
    private final WebClient webClient;
    
    @Value("${opencloud.monitor.prometheus-url}")
    private String prometheusUrl;
    
    // 瞬时查询
    public Mono<PromQueryResult> query(String promql, Long time) {
        return webClient.get()
            .uri(uriBuilder -> uriBuilder
                .path("/api/v1/query")
                .queryParam("query", promql)
                .queryParamIfPresent("time", Optional.ofNullable(time))
                .build())
            .retrieve()
            .bodyToMono(PromQueryResult.class);
    }
    
    // 范围查询
    public Mono<PromQueryResult> queryRange(String promql, Long start, Long end, Long step) {
        // 校验时间范围 ≤ 30 天
        if (end - start > 30 * 24 * 60 * 60) {
            throw new MonitorException("查询时间范围不能超过 30 天");
        }
        return webClient.get()
            .uri(uriBuilder -> uriBuilder
                .path("/api/v1/query_range")
                .queryParam("query", promql)
                .queryParam("start", start)
                .queryParam("end", end)
                .queryParam("step", step)
                .build())
            .retrieve()
            .bodyToMono(PromQueryResult.class);
    }
}
```

### 5.3 Kubernetes 对接

```java
// Kubernetes 客户端（opencloud-cloud）
@Component
@RequiredArgsConstructor
public class KubernetesClient {
    
    private final ClusterConfigService configService;
    
    // 获取 Fabric8 客户端
    public KubernetesClient getClient(Long clusterId) {
        ClusterConfig config = configService.getById(clusterId);
        String kubeConfig = decrypt(config.getKubeConfig());
        
        Config fabric8Config = Config.fromKubeconfig(kubeConfig);
        return new KubernetesClientBuilder()
            .withConfig(fabric8Config)
            .build();
    }
    
    // 列出命名空间
    public List<NamespaceVO> listNamespaces(Long clusterId) {
        try (KubernetesClient client = getClient(clusterId)) {
            return client.namespaces().list().getItems().stream()
                .map(this::convertToVO)
                .collect(Collectors.toList());
        }
    }
}
```
