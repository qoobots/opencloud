## 五、索引设计原则

### 5.1 索引创建规范

1. **主键索引**：所有表使用 BIGSERIAL 自增主键
2. **唯一索引**：业务唯一字段（如 username + tenant_id）
3. **外键索引**：关联字段自动创建索引
4. **查询索引**：高频查询条件字段
5. **复合索引**：多条件查询场景

### 5.2 索引命名规范

```
idx_{table_name}_{column_name}           -- 单列索引
idx_{table_name}_{col1}_{col2}           -- 复合索引
idx_{table_name}_{column}_tenant         -- 租户隔离索引
```
