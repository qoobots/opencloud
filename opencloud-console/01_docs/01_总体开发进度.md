## 项目概述

OpenCloud 控制台是面向云计算基础设施的统一管理平台，支持 Ceph 存储、OpenStack 云主机、Kubernetes 容器编排、Prometheus/Grafana 监控等核心组件的可视化管理。

**技术栈**

| 层次 | 技术选型 |
|------|----------|
| 后端框架 | Spring Boot 3.2 + Java 17 + Maven 多模块 |
| 数据库 | PostgreSQL 16 + pgvector（预留 AI 向量检索） |
| 缓存 | Redis 7 |
| ORM | MyBatis-Plus |
| 安全 | Spring Security + JWT（Access Token + Refresh Token + Redis 黑名单） |
| 前端框架 | Vue 3 + Vite 5 |
| UI 组件库 | Element Plus |
| 状态管理 | Pinia |
| 图表 | ECharts |
| HTTP 客户端 | Axios（含 Token 自动刷新） |

## 开发环境启动

### 1. 启动基础服务

```bash
cd opencloud-console
docker-compose up -d
```

### 2. 初始化数据库

```bash
# 连接 PostgreSQL 执行初始化 SQL
psql -h localhost -U opencloud -d opencloud_console \
     -f opencloud-web/src/main/resources/db/init.sql
```

### 3. 启动后端

```bash
mvn clean package -DskipTests
java -jar opencloud-web/target/opencloud-web-1.0.0.jar \
     --spring.profiles.active=dev
```

### 4. 启动前端

```bash
cd opencloud-ui
npm install
npm run dev
# 访问 http://localhost:3000
```
