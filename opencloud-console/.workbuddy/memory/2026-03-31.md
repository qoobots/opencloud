## opencloud-cloud v1.1.0 完善功能版 (2026-03-31)

### ✅ v1.1.0 新增内容
1. **OpenStack 镜像上传**：`OpenStackImageService.uploadImage()` + `OpenStackImageController.uploadImage()`
   - 支持分块上传
   - 支持容器格式/磁盘格式配置
   - 支持最小磁盘/内存配置

2. **Ceph MGR REST API**：完善 `CephServiceImpl`
   - 集群总览：`getOverview()` 调用 MGR API
   - 存储池列表：`listPools()` 调用 MGR API
   - 存储池详情：`getPool()` 调用 MGR API
   - OSD 列表：`listOsds()` 调用 MGR API
   - 添加 `CloudClientFactory.getClusterConfig()` 方法

3. **Kubernetes Pod 日志流**：`KubernetesService.getPodLogsStream()` + `KubernetesController.getPodLogsStream()`
   - 支持 SSE 流式日志输出
   - 支持 follow 模式持续输出

4. **配额设置接口**：`CloudQuotaService.getQuota()` / `setQuota()` + `CloudQuotaServiceImpl`
   - 支持查询租户配额
   - 支持创建/更新租户配额

5. **定时同步任务**：`CloudSyncJob.java`
   - 每小时同步集群状态
   - 每天凌晨 2 点同步资源使用量
