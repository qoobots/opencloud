# opencloud-common — 开发进度

> 模块职责：公共基础层，提供统一响应体、全局异常处理、工具类、基础实体等，被所有其他模块依赖。
>
> 最后更新：2026-03-31

---

## 功能清单

### 统一响应

| 类 | 描述 | 状态 |
|----|------|------|
| `R<T>` | 统一响应体（code / msg / data / timestamp） | ✅ 完成 |
| `ResultCode` | 业务状态码枚举（SUCCESS / UNAUTHORIZED / FORBIDDEN / NOT_FOUND / BUSINESS_ERROR / SYSTEM_ERROR） | ✅ 完成 |

### 异常处理

| 类 | 描述 | 状态 |
|----|------|------|
| `BusinessException` | 业务异常，携带 ResultCode | ✅ 完成 |
| `GlobalExceptionHandler` | `@RestControllerAdvice`，统一处理 Validation / Security / Business / 未知异常 | ✅ 完成 |

### 基础实体

| 类 | 描述 | 状态 |
|----|------|------|
| `BaseEntity` | 公共字段：id / createTime / updateTime / createBy / updateBy / deleted（逻辑删除） | ✅ 完成 |
| `MybatisPlusMetaHandler` | `@TableField(fill)` 自动填充时间与操作人 | ✅ 完成 |

### 安全工具

| 类 | 描述 | 状态 |
|----|------|------|
| `JwtUtil` | Access Token / Refresh Token 生成、解析、校验（jjwt） | ✅ 完成 |
| `SecurityUtil` | 从 `SecurityContextHolder` 获取当前登录用户 | ✅ 完成 |
| `LoginUser` | Spring Security `UserDetails` 实现，携带用户ID、角色、权限列表 | ✅ 完成 |

## 注意事项

- 本模块**不包含** `@SpringBootApplication`，不可独立启动
- 所有子模块通过 `pom.xml` 引入本模块：`<dependency>opencloud-common</dependency>`
- `BaseEntity` 中的 `createBy` / `updateBy` 依赖 `SecurityUtil.getCurrentUserId()`，需确保 Security 上下文已就绪
