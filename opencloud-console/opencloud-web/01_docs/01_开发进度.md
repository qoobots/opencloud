## 注意事项

- **安全**：生产部署时 `JWT_SECRET` 和 `ENCRYPT_KEY` 必须使用强随机密钥，绝不使用默认值
- **CORS**：生产环境 `ALLOWED_ORIGINS` 必须指定明确域名，禁止使用 `*` 通配符
- **Flyway**：迁移脚本命名规范 `V{major}.{minor}.{patch}__{description}.sql`，生产环境禁止 `flyway clean`
- **Actuator**：`/actuator/prometheus` 端点在生产环境建议配置 Nginx IP 白名单限制访问
- **日志**：使用 `logback-spring.xml` 区分环境，生产日志滚动归档，traceId 贯穿全链路
- **优雅停机**：`server.shutdown=graceful` + `lifecycle.timeout-per-shutdown-phase=30s`，确保服务重启时正在处理的请求完成后再退出
